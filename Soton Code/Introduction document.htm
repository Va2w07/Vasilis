Transfer Function based refractive index extraction algorithm for THz-TDS

 

 

This algorithm is working based on transfer function in order to extract a refractive index from a slab material. It works in the context of a THz time domain spectroscopy measurements that have been taken in transmission geometry. As input the algorithm needs a time domain scan of a sample and a time domain scan of a reference scan, also the thickness of the sample. The algorithm will give as output the complex refractive index of the sample.

 

•     The complex refractive index of a material is determined by fitting a theoretically calculated transfer function to the experimental data.

•     The calculation of the theoretical transfer function is done using Fresnel coefficients

•     The calculation of the experimental transfer function is the division in frequency of the signal obtained with the same and a reference scan without the sample

•     Complex refractive index can be used to determine material properties such as relative permittivity, conductivity
